# 날아라 물리 통합과학 — 수업자료 HTML 템플릿 가이드

> 이 파일은 새로운 단원 HTML을 동일한 디자인으로 제작하기 위한 작성 가이드입니다.
> 아래 **[작성 내용 입력 시트]** 를 채운 뒤, Claude에게 붙여넣으면 완성된 HTML을 받을 수 있습니다.

---

## 사용 방법

1. 아래 **[작성 내용 입력 시트]** 를 복사합니다.
2. 각 항목에 새 단원의 내용을 채웁니다.
3. Claude에게 다음 메시지와 함께 붙여넣습니다:

```
아래 시트를 보고, "통합과학_수업자료_템플릿_가이드.md"의 디자인 규칙에 따라
완성된 HTML 파일을 만들어 줘.
```

---

## [작성 내용 입력 시트]

```
=== 기본 정보 ===
파일명        : 예) 과학의 기초_물질과 에너지.html
브라우저 탭 제목 : 예) 통합과학 수업자료 | 물질과 에너지
헤더 배지     : 예) 통합과학 1 · 과학의 기초
헤더 대제목   : 예) 물질과 에너지의 관계
헤더 소개문   : 예) 물질과 에너지의 전환 원리를 이해하고 일상 속 사례를 탐구해봅시다.

=== 1. 생각 열기 (연노랑 배경) ===
도입 문장     : 예) 에너지는 어디서 와서 어디로 갈까요?

사례 1 제목   : 예) 태양광 전지
사례 1 설명   : 예) ...
사례 1 세부내용: 예) ...

사례 2 제목   : 예) 음식과 체온
사례 2 설명   : 예) ...
사례 2 세부내용: 예) ...

핵심 질문     : 예) 에너지는 형태가 바뀔 뿐 사라지지 않는다는 것이 왜 중요할까요?

=== 2. 핵심개념 익히기 (연파랑 배경) ===
개념 소단원 목록 (각 소단원마다 아래 형식 반복):

  [소단원 1]
  제목       : 예) 에너지의 종류
  설명       : 예) 에너지는 운동 에너지, 위치 에너지, 열에너지, 전기 에너지 등 다양한 형태로 존재합니다.
  칩(태그) 목록: 예) 운동 에너지, 위치 에너지, 열에너지, 전기 에너지, 빛에너지
  표 필요 여부: 예) 있음 / 없음
  표 내용(있을 경우):
    열제목: 예) 에너지 종류 | 단위 | 예시
    행 1  : 운동 에너지 | J | 달리는 공
    행 2  : 위치 에너지 | J | 높은 곳의 물체
    행 3  : 열에너지   | J | 뜨거운 물

  [소단원 2]
  제목       :
  설명       :
  칩 목록    :
  표 필요 여부:
  표 내용    :

  (소단원 추가 시 계속 반복)

안내 박스(notice) 내용: 예) • 에너지 보존 법칙: 에너지는 형태가 바뀔 뿐 총량은 일정하다.

=== 3. 일상과 연결하기 (연초록 배경) ===

  [활동 1]
  제목       : 예) 에너지 전환 경로 추적하기
  설명       : 예) 다음 상황에서 에너지 전환이 어떻게 일어나는지 빈칸을 채워봅시다.
  빈칸 문장 목록:
    - "스마트폰을 충전한다." → (     ) 에너지 → (     ) 에너지
    - "밥을 먹고 걷는다."   → (     ) 에너지 → (     ) 에너지

  [활동 2]
  제목       : 예) 뉴스 탐구
  뉴스 제목  : 예) [뉴스] 태양광 발전소 화재 사고
  뉴스 본문  : 예) ...

  [활동 3]
  제목       : 예) 해결 아이디어 제안하기
  설명       : 예) 에너지 낭비를 줄이기 위한 아이디어를 제안해 보세요.
  칩 목록    : 예) 단열재 강화, 회생제동 시스템, 스마트 전력 관리

핵심 정리 항목:
  - 예) 에너지는 여러 형태로 전환될 수 있다.
  - 예) 에너지의 총량은 항상 보존된다.
  - 예) 일상 속 기술은 에너지 전환 원리를 응용한 것이다.

=== 4. 형성평가 (연보라 배경) ===
객관식 문제 (최대 5문제, 각 4지선다):

  Q1. 문제: 예) 다음 중 에너지의 단위로 옳은 것은?
      ① W  ② J  ③ m/s  ④ kg
      정답번호: 2

  Q2. 문제:
      ① ② ③ ④
      정답번호:

  Q3. 문제:
      ① ② ③ ④
      정답번호:

  Q4. 문제:
      ① ② ③ ④
      정답번호:

  Q5. 문제:
      ① ② ③ ④
      정답번호:

주관식 문제 (1문제):
  Q6. 문제: 예) 에너지가 형태는 바뀌지만 총량은 변하지 않는 법칙을 무엇이라 하는지 쓰시오.
      정답 : 예) 에너지 보존 법칙
```

## 디자인 규칙 (Claude가 참고하는 사양)

### 색상 팔레트

| 변수 | 값 | 용도 |
|---|---|---|
| `--primary` | `#2b6cb0` | h2 제목, 버튼 |
| `--accent` | `#0f766e` | h3 소제목 |
| `--primary-soft` | `#dbeafe` | 키박스, 표 헤더 |
| `--bg` | `#f4f7fb` | 페이지 배경 |

### 섹션별 배경색

| 섹션 | 클래스/ID | 배경색 |
|---|---|---|
| 1. 생각 열기 | `.hero-card` | `#fffdf0` (연노랑) |
| 2. 핵심개념 익히기 | `#sec-concept` | `#eff6ff` (연파랑) |
| 3. 일상과 연결하기 | `#sec-daily` | `#f0fdf4` (연초록) |
| 4. 형성평가 | `.quiz-card` | `#fdf4ff` (연보라) |

### 주요 컴포넌트

| 컴포넌트 | 클래스 | 용도 |
|---|---|---|
| 키워드 태그 | `.chip` | 개념 목록 나열 |
| 펼침 사례 버튼 | `.case-button` + `.case-detail` | 클릭 시 내용 토글 |
| 활동 박스 | `.activity-step` | 학생 활동 지시문 |
| 안내 박스 | `.notice` | 주의·핵심 강조 |
| 핵심 정리 | `.summary-box` | 단원 마무리 요약 |
| 빈칸 | `.fill` | 빈칸 채우기 밑줄 |
| 키박스 | `.keybox` | 핵심 질문 배경 강조 |

### 헤더 그라데이션
```css
background: linear-gradient(135deg, #1d4ed8 0%, #0f766e 100%);
```
단원 성격에 따라 색상 조합 변경 가능 (예: 생물 → 초록 계열, 화학 → 주황 계열).

### 푸터 고정 문구
```html
날아라 물리 통합과학 학습자료 | © 동북고등학교 강현식
```

### 자바스크립트 동작
- **사례 토글**: `toggleCase('id')` — `.case-detail`에 `.show` 클래스 추가/제거
- **채점**: `gradeQuiz()` — 객관식 정답 객체 + 주관식 문자열 비교
- **리셋**: `resetQuiz()` — 모든 입력 초기화
